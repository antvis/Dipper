# dipper

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```

### 新增加 LIB

packages 目录下新建目录名,在新建目录下执行创建项目

`yarn create @umijs/dumi-lib`
