## 简介

[Dipper](https://github.com/antvis/dipper) 是基于 L7 的地理分析应用开发框架，用于快速构建和开发地理分析应用。用户通过组件化、模块化、低代码的方式配置开发地图可视化分析应用。

## 特性

- 简单易用、快速上手
- 内置常用地图分析组件
- 一切皆组件、灵活注册、配置组件
- 定制扩展开发，只需关注业务逻辑
