# GridLayer 填充图

## 填充图

填充图主要用展示，区域类，网格类的面状数据的可视化展示。

### 构造函数

### name

    网格名称、唯一值，可根据网格名称获取网格实例

### data

    网格数据，数格式为 GeoJson,目前为开放其他数据扩展

### options

    图层配置项目，用于设置图层样式，如标注字段，颜色，填充颜色字段的部分的

### 方法

#### show

图层选

#### hide

图层隐藏

#### destroy

图层销毁
